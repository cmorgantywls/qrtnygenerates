INSTRUCTIONS

Move the mouse to change the number of tiles (rects) being displayed in rows and columns. Press 1 or 2 to change the interpolation from passing through RGB mode or straight HSB mode. Click to reroll the color palette using shakeColors() function. Click 's' to save image of canvas.

TAKE AWAYS

  • Utilization of map to control the number of shapes drawn based on mouse position - useful application for students learning map.
  • Ways to showcase the different colors generated by RGB and HSB by using both in an algorithmic sequence.
  • Saving image of canvas based - you can designate time stamp and correct file extension.
  • lerpColor == performs interpolation between individual color values. Amount specifies the position between the start and end of the color.
